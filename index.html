Projects

Game Your Fit
ANDROID FIREBASE WEBRTC KOTLIN
Top 10 finalist for Google Solution Challenge 2021. Game Your Fit is a workout application that's designed to motivate users to exercise more through gamification. Users can open the website and control the player character using sensors on their phone. The user's movement will be tracked and sent as an input to the browser.

This project was a 6-month long marathon of us learning many new technologies. I was responsible for designing and coding the Android UI and backend APIs. One of the unique challenge that we faced in this project is our relatively small team size. Our big scope combined meant that every member had to wear many hats and dip their toes in different parts of the projects. Other than the Android application, I also helped other members in designing the WebRTC interface to communicate between devices.

Wonder Reader
TYPESCRIPT ANDROID GCP C++ NGINX
Top 10 finalist for Google Solution Challenge 2023. A tool to teach braille to visually impaired people. This project is composed of both hardware and software, all of it custom-made by our team. With this device, a teacher could help a student learn braille using the interface of their phone, where the student would then respond using the device.

One of the project with the most changes in its lifespan. Originally, it started as a hardware to interface with the internet using braille. It then changed to a web-based teaching device, ending up on an offline Android application.
The reason for these changes is due to our team's insistence on fulfilling a real-world need using the application. We repeatedly tested this on the local visually impaired organization, making sure we were on the right track.

Overall, the most fulfilling project that I have ever done, with the most amount of repositories.
Translate-inator
TYPESCRIPT ANGULAR GCP DOCKER NGINX
Full-stack project of a note taking web application with a translation function. It is currently deployed on this webpage. Deployed on a Google Compute Engine using Docker compose and nginx reverse proxy. This is a solo project that uses Google's Cloud Translation API. The backend is created using Typescript and Node.js, while the frontend uses Angular with rxjs. Deployment is handled with Github Action CI/CD.

This project posed a different type of challenge in how I approached my development. I went into it with the mindset of "This should be maintainable by a complete stranger", which proved to be more difficult than I thought. Coding the functionality itself was not a problem, but everything surrounding it had to be easy to modify and should allow customization for different environments (production, development, etc.).

The final result of this 13-day sprint is a full-stack application that can be run with two docker commands (assuming you have provided the correct .env variable) 

docker compose -f ./docker-compose.setup.yaml up -d
docker compose up -d
